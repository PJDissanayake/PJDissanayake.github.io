# Pathum Dissanayake - Portfolio Website

Modern, professional portfolio website showcasing embedded systems expertise and EV fast-charging projects.

## Features

- Dark theme with glassmorphism effects
- Animated particle.js background
- Fully responsive mobile-first design
- Smooth scroll animations
- Project filtering system
- Contact form with mailto integration

## Technologies Used

- HTML5, CSS3, JavaScript
- Particles.js for animated background
- AOS (Animate On Scroll) library
- Google Fonts: Inter, JetBrains Mono, Space Grotesk

## Setup for GitHub Pages

1. Upload your resume PDF as `Pathum_Dissanayake.pdf` to the repository root
2. Update social media links in `index.html`:
   - LinkedIn: `https://linkedin.com/in/pathum-dissanayake`
   - GitHub: `https://github.com/pathum-dissanayake`
   - YouTube: `https://youtube.com/@pathum-embedded`
3. Push to GitHub and enable GitHub Pages in repository settings

## Local Development

Simply open `index.html` in a web browser. No build process required.
